# Kolam Classification using Hybrid CNN + Vision Transformer 🧠🎨

This project focuses on classifying Kolam patterns using a hybrid deep learning model that combines Convolutional Neural Networks (CNN) and Vision Transformers (ViT).

Kolam is a traditional South Indian art form with intricate geometric patterns, and this project aims to apply modern deep learning techniques to recognize and classify these designs.

---

## 📌 Overview

- Hybrid architecture: CNN + Vision Transformer
- Built using PyTorch
- Supports training, evaluation, and single-image prediction
- Includes data augmentation for improved generalization

---

## 🧠 Model Architecture

This project uses a hybrid approach:

- **CNN** → extracts local spatial features (edges, textures)
- **Vision Transformer (ViT)** → captures global relationships and patterns
- Combined model leverages both local and global feature understanding
