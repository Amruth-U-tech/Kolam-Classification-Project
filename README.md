This project focuses on classifying Kolam patterns using deep learning techniques.
Kolam is a traditional South Indian art form, and this project aims to apply computer vision to recognize different designs.

## Motivation

Kolam patterns are culturally significant but complex in structure.
This project explores how deep learning can be used to understand and classify these intricate patterns.

The goal is to combine cultural heritage with modern AI techniques.

## Tech Stack

- Python
- Google Colab (GPU)
- TensorFlow / PyTorch
- NumPy

  ## Methodology

- Data preprocessing
- Feature extraction
- Model training (CNN)
- Evaluation and prediction
